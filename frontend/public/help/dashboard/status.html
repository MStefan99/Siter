This section tells you whether the Siter web server is functioning properly. The purpose of the server is to serve client requests by either sending static files or forwarding them to the destination servers depending on the configuration. There are three possible options:

  • Green, server is available and working properly
  • Yellow, server is available but not functioning properly
  • Red, server unavailable

If there are any problems with the server, you will see a message with possible steps for solving the problem.
