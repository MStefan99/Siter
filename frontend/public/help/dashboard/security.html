Security section is used by apps to protect the users of your websites from attackers. You will find a few options here.

Hosting your sites over insecure HTTP protocol is dangerous since it has no security and the data sent via HTTP will be visible for other users. Using HTTP is only recommended for testing purposes.

To enable HTTPS, a secure version of HTTP, check the Enable HTTPS checkbox when creating or editing a route. You will need to get a certificate to serve your site over HTTPS. Certificate authority will usually provide you with two files: a certificate file and a key file. Save these files and use an integrated file picker to select them.
