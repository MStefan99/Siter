Siter consists of two parts: a server running on your device and a web interface which you're using now. These parts are independent so this section tells you whether the Siter web server is functioning properly. There are three possible statuses:

  • Green, server is available and working properly
  • Yellow, server is available but not functioning properly
  • Red, server unavailable

If there are any problems with the server, you will see a message with possible steps for solving the problem.
